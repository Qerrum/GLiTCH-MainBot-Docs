Discord Moderator Commands

Last updated: 26/08/2019
If you encounter issues please contact Qerrum#6886 on Discord.
Command Usage Description
-mute <@user> [time] [reason] -mute @Qerrum#6886 10m Spamming Mute a user based on given time and with a reason.
Time; s=seconds/m=minutes/h=hours/d=days
-purge [amount] -purge 50 Purge the channel of messages based on the amount given.
-kick [@user] [reason] -purge @Qerrum#6886 Too much spamming Kick a user from the server.
Please note that the command needs to be used where the user is taggable (#pub or #bot-commands)
-warn [@user] [reason] -purge @Qerrum#6886 Breaking rule #1 Warns a user, pms them the reason.
-clearwarn [@user] -purge @Qerrum#6886 Breaking rule #1 Removes all warnings from a user.
-warns [@user] -purge @Qerrum#6886 Shows if they have any active warnings.
-ban [@user] [reason] -ban @Qerrum#6886 Multiple warns, rules broken Bans the user from the server.
Do not use this as it is currently broken
-reload [command] -reload avatar Reload the cache of a command.
Do not use this as it is currently broken
-lockdown [#channel] [time] -lockdown #pub 10m Locks the specified channel, for [time] if specified else forever.
-unlockdown [#channel] -lockdown #pub Undoes what -lockdown does.
-approve [msg id] [reason] -lockdown 616218033379082240 Good suggestion, etc.. Approve a suggestion from a user. Make sure the msg id is the suggestion message id.
-deny [msg id] [reason] -lockdown 616218033379082240 Not approved due to, etc.. Deny a suggestion from a user. Make sure the msg id is the suggestion message id.
-emojilist -emojilist List all the emojis of the server in a nice way. Taxing on the bot so dont over do it.
